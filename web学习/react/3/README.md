Building
--------

```bash
git clone git@github.com:eveningwater/my-web-projects/react/3.git
cd my-web-projects/react/3
npm install
npm run dev
# Open localhost:8080/index-dev.html in your browser
```

Building for Production
--------

```bash
npm run build
```
